### NoteApp – Simple Notes Taking Application (Spring Boot + JSP + Swagger)

NoteApp is a simple and lightweight notes-management application built using Spring Boot, JSP, and MySQL.  
It allows users to add, update, delete, and view notes.  
Swagger UI is included for testing all API endpoints easily.

### Project Overview

- A full-stack CRUD application for recording notes.
- Developed using Spring Boot with MVC architecture.
- JSP is used for server-side user interface pages.
- MySQL is used as the persistent database.
- Swagger/OpenAPI is integrated for testing APIs.
- Simple and clean UI suitable for beginners.
- Follows a proper controller → service → repository structure.

### Features

- Add a new note
- Update the existing note
- Edit the title of a note
- Delete a note
- View all notes
- API documentation available through Swagger
- Easy to extend for additional features

### Tech Stack

- Backend: Spring Boot, Spring MVC, JPA
- Frontend: JSP, HTML, CSS
- Database: MySQL
- Documentation: Swagger / OpenAPI
- Build Tool: Maven
